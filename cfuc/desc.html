Description

FDCAN USB Converter (CFDC) works as CAN to USB converter. Multiple features are supported like:

  • CAN Flexible Data Rate version 1.0 (CAN FD) supported, Microchip MCP2542FD FD CAN Transceiver
  • CAN version 2.0 support (ISO 11898-1: 2015)
  • receiving and sending of normal and extended frames, brs frames
  • CAN baudrate up to 8 Mbps
  • device size is 26x14 mm (including USB connector)
  • selectable termination resistors
  • python-can support
  • ***currently not supported *** internal/external loop back mode, listen only mode
  • ***currently not supported *** hardware frames filtering

    • See tutorial on YouTube

    Drivers

    After USB enumeration device creates virtual COM port that is used both for data transfer and converter setup. Device is powered directly from USB

    uCAN ViewerV2

    For basic interaction with device there is precompiled python application uCANViewerV2. This application lets you receive and send and log FD and Classic CAN frames. For detailed information see uCANViewerV2 For CAN interface configuration see assets/can.conf
