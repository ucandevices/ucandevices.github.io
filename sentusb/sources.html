Hardware

The device uses an STM32F042K6 microcontroller running at 48 MHz from the internal HSI48 oscillator — no external crystal required. PCB is designed in KiCad. Board size is 17×28 mm. All hardware source files are available on GitHub: here.

Software

Firmware is written in C using STM32CubeIDE (free). It uses the HAL USB CDC driver and custom SENT input-capture / output-compare logic on TIM2 and TIM14. All sources and releases are available on GitHub: here.

Uploading new firmware

The device has an embedded USB DFU bootloader. Use dfu-util to flash new firmware: 

dfu-util -a 0 -s 0x08000000 -D firmware.bin
On Windows you can also use ST DfuSe tool.
